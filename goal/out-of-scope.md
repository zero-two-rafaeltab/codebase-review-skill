# Out of Scope

This document is historical/contextual goal guidance. The repository now includes
the initial Hermes skill package, so this list describes what remains outside the
current focused manual discovery workflow rather than what is impossible to add
to the repository.

The following remain out of scope for the current workflow:

- creating reference documents beyond the minimal skill guidance;
- creating templates;
- creating install scripts;
- creating validation scripts;
- creating language playbooks;
- installing the skill into a user's local Hermes profile;
- setting up review tooling in any target repository;
- adding CI integration;
- opening GitHub issues based on the goal;
- writing ADRs based on the goal;
- performing codebase changes as part of discovery.
